## [1.0.0] - 2025-12-23

### Added
- Initial public release
- Spell-check enabled DataFlex controls extending:
  - `cTextEdit`
  - `cDbTextEdit`
- Windows DLL wrapper using the native Windows Spell Checking API (`ISpellCheckerFactory`)
- Support for both 32-bit and 64-bit DataFlex applications
